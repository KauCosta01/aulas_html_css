ID & CLASS

ID

O ID funciona como um CPF. Assim como só posso ter um número de CPF, cada elemento só pode ter um valor de ID. O CPF é único, ou seja, ninguém pode ter um CPF igual ao meu. Isso significa que o ID também é único, então não é possível repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID, Diferente do ID em que você só pode tem um valor, no CLASS você pode ter mais de um. Também diferente do ID, em que não se pode repetir um valor, no CLASS vocè pode utilizar o mesmo valor para vários elementos.
